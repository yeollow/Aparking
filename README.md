## 🚗 아파트·상가의 주차면 공유서비스 플랫폼
* 주차면 예약·결제를 위한 안드로이드 애플리케이션
* 특정 시간대에 남는 아파트의 주차 공간을 외부인에게 공유
* 주차공간의 수급 불균형 해소 및 공유경제 활성화
* 불법 주정차 사고 예방 및 공동체 의식 제고
